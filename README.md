# 🏥 دکتر رزرو - سیستم نوبت‌دهی آنلاین پزشکی

![دکتر رزرو](public/images/Logo%20(1).jpg)

## 📋 درباره پروژه
پروژه **دکتر رزرو** یک سیستم نوبت‌دهی آنلاین برای پزشکان است که کاربران می‌توانند به راحتی پزشک مورد نظر خود را پیدا کرده، نوبت رزرو کنند و نظرات خود را ثبت نمایند.

---

## 🛠 تکنولوژی‌های استفاده شده

### Frontend
- **Next.js** - فریمورک React
- **TypeScript** - زبان برنامه‌نویسی
- **Tailwind CSS** - استایل‌دهی
- **Vazirmatn** - فونت فارسی

### Backend
- **Next.js API Routes** - سمت سرور
- **MongoDB** - دیتابیس
- **Mongoose** - ODM
- **JWT** - احراز هویت
- **Bcrypt** - هش کردن رمز عبور

---
